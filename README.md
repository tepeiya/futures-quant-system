# 期货网格量化系统 v3 - 量子女王增强版

## 三大核心改进

### 1️⃣ 大周期趋势过滤 (周线级)
- 55日EMA > 120日EMA = 牛
- 55日EMA < 120日EMA = 熊
- 价格在55日线上下决定方向
- 只在EMA多头排列时做多，空头时做空

### 2️⃣ 波动率过滤器
- ATR/价格 < 0.2% 或 > 8% → 不开仓
- ATR相对20日均值 < 0.5 或 > 3.0 → 不开仓
- 避免"僵尸行情"和"崩盘行情"

### 3️⃣ 市场状态识别 (Rule-based)
- **效率比(ER)**: 方向性移动/总波动 → 判断趋势强弱
- **ADX**: 多空力量差异度
- **价格-EMA距离**: 远离均线 = 趋势中
- 趋势市→正常网格,震荡市→单仓小利就走

## 策略逻辑
```
趋势市 → 正常网格(多仓) + 分层部分止盈(50%) + 宽松移动止损
震荡市 → 只开1仓 + 1%盈利全平 + 严格止损
波动过低 → 不开仓
趋势反转 → 平反向仓
```

## 用法

```bash
cd /var/minis/shared/futures-grid/

# 回测白银
python3 backtest/runner.py --symbol AG --months 12 --balance 100000 --volume 1

# 回测黄金 (保守参数)
python3 backtest/runner.py --symbol AU --months 24 --volume 0.3 --grid 1.5 --max-pos 2

# 回测螺纹钢
python3 backtest/runner.py --symbol RB --months 12 --balance 50000 --volume 2

# 参数说明
# --symbol: AU(金) AG(银) RB(螺纹) CU(铜) SC(原油) I(铁矿) ...
# --months: 回测月数
# --volume: 首单手数 (取决于品种和资金)
# --grid:   网格间距(ATR倍数),越大越不容易加仓,默认1.5
# --max-pos: 最大持仓层数,默认4
# --sl:     止损(ATR倍数),越大越宽松,默认4.0
```
